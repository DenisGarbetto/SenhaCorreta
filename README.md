# SenhaCorreta

Programa simples em Java para validação de login com usuário e senha fixos.

## Sobre o projeto
Este projeto consiste em uma aplicação console que solicita ao usuário um nome de usuário e senha, e só libera acesso quando ambos estiverem corretos. É uma implementação básica utilizando laço de repetição e comparação de Strings.

## Funcionalidades
- Solicitação de usuário e senha via teclado
- Validação dos dados informados com valores pré-definidos
- Repetição do pedido até as credenciais corretas serem inseridas
- Mensagem de acesso liberado após autenticação bem-sucedida

## Como executar
1. Clone este repositório
2. Compile o programa com `javac SenhaCorreta.java`
3. Execute com `java SenhaCorreta`
4. Digite o usuário: `Denis`
5. Digite a senha: `1234`
6. Caso informações estejam erradas, o programa solicitará repetir até sucesso.

## Tecnologias utilizadas
- Java (JDK)
- Scanner para leitura de entrada via console
